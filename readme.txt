This plugin is a collection of custom email templates and functionality for STAY FOXHOLE, LLC, built for use with HivePress.

== Description ==

A custom plugin that extends HivePress by providing branded emails for the STAY FOXHOLE platform. It allows sending tailored messages to users for key actions like registration, booking, and vendor communication.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate through the 'Plugins' menu in WordPress.
3. Configure emails under HivePress settings (if applicable).
